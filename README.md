# ☕ Lamiz Coffee Website Redesign

## 📌 About the Project

This is a university project in which I redesigned the **Lamiz Coffee** website with a modern, clean, and fully responsive user interface.

The project consists of **8 pages**, including:

* Home
* Menu
* Branches
* About Us
* Contact Us
* Login
* Shopping Cart
* Other supporting pages

The **Menu** page dynamically displays products using an API. Users can browse different categories of drinks, desserts, and sandwiches, then add or remove products from their shopping cart. The shopping cart is managed using **Local Storage**, allowing users to keep their selected items during their session.

The project was developed with a strong focus on creating an intuitive user experience, responsive layouts, and interactive functionality across all pages.

---

## ✨ Features

* Fully Responsive Design
* Modern and Clean UI/UX
* Dynamic Menu Using API
* Product Categories
* Add & Remove Products from Shopping Cart

## 🚀 How to Run

1. Clone the repository.

```bash
git clone <repository-url>
```

2. Install JSON Server.

```bash
npm install -g json-server
```

3. Run the API.

```bash
json-server --watch db.json --port 3000
```

4. Open the project using **Live Server** or any local web server.
